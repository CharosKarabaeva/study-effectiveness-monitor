# Техническое задание

## Назначение системы
Разработка веб-сервиса для анализа учебной эффективности
на основе психоэмоциональных параметров обучающегося.

## Функциональные требования
- добавление учебных дней;
- учет настроения, усталости и продуктивности;
- хранение данных в базе данных;
- фильтрация по датам;
- визуализация данных;
- формирование рекомендаций;
- классификация учебных дней.

## Структура данных
Сущность StudyDay:
- date
- mood
- fatigue
- productivity
- comment

## Пользователи
- обучающийся

## Технологии
- Django
- Python
- HTML / CSS
- Bootstrap
